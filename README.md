# Kundenverwaltung

## Beschreibung
Dieses Projekt ermöglicht das Speichern und Löschen von Kundendaten.

## Features
- Kunden hinzufügen
- Kunden löschen

## Installation
1. Repository klonen:
   ```bash
   git clone https://github.com/AzlanGitHub/java.git
   ```
2. In das Projektverzeichnis wechseln:
   ```bash
   cd projekverzeichnis

   ```
3. Abhängigkeiten installieren (falls erforderlich):
   ```bash
   mvn install
   ```

## Nutzung
1. Anwendung starten:
   ```bash
   java -jar kundenverwaltung.jar
   ```
2. Kunden hinzufügen oder löschen über die Benutzeroberfläche oder API.

## Voraussetzungen
- Java 17+
- Maven
- SQLite
